# Changelog

## 2.0.0 - 2026-04-23
- Android 客户端完成第二版功能与 UI 收口，补齐底部玻璃拟态导航、消息中心收口、课程订阅语义和多角色首页交互修正。
- 正式版本号提升为 `versionCode 2`、`versionName 2.0.0`，可直接生成第二版 APK 产物。
- 新增 Android 项目打包说明书，现包含 `demo`、`prod` 两种变体、签名配置和输出路径说明。

## 1.2.0 - 2026-04-22
- Android 登录页移除演示账号快捷入口，登录工作台下沉到底部，并补充企业材料上传与注册拦截逻辑。
- Android 端补齐消息中心、帮助与反馈、课程管理、禁飞区与飞行申请等业务闭环说明。
- 后端确认以 `MyBatis-Plus + Flyway + MySQL` 持久化实现核心业务，并补充管理端聚合接口说明。
- 管理端确认移除 `PageContainer.vue` 返回按钮，并记录 `/api/admin/users` 空值兜底修复。
- 更新根目录与 `docs` 文档索引，同步项目说明书、架构、接口、数据库、测试、启动与验收文档。

## 1.1.0 - 2026-04-22
- 建立 Sonar、JaCoCo、Vitest 覆盖率与多作业 CI 流水线。
- 引入 Helm/Kubernetes、镜像签名、部署/回滚/备份脚本与告警配置。
- 后端将刷新令牌、审计事件与风险告警迁移至持久化存储。
- Android 端增强任务缓存时效控制与统一错误提示。
- 补齐需求、架构、运维、用户手册与灾备文档。
